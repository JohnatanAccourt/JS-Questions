this

O this é uma variável que depende do contexto, escopo e do modo restrito ele pode ter resultados diferentes

se temos this global e uma dentro da função obviamente vamos ter resultados diferentes porque são contextos diferentes o this dentro da função tem seu escopo diferente, quando é uma arrow function a variável this vai continuar sendo global e com isso sempre terá um contexto de onde ele veio sendo assim não vai ter aquele scopo de função ele vai sempre enxergar o contexto.

a ECMASCRIPT trouxe o bind que permite com que o contexto nunca se perca, ou seja, quando ele é desconectado ou desacoplado de um objeto ele perde a conexão então com o bind e colocando a referencia que no caso é o objeto dele mesmo ele volta a se conectar. A ECMA também trouxe as arrow functions que como mencionado permite com que o contexto se mantenha global dentro de uma função ou então quando é declarado dentro de um objeto a função
